용어사전 — 방사선 방호 & GEANT4 핵심 용어

ShieldDose 프로젝트에서 사용하는 핵심 용어 35개. ISS/PIT/LT 항목과 연계하여 이해를 돕기 위한 참고 자료.

방사선 물리 (8)

CPE (하전입자평형)

Charged Particle Equilibrium

임의 체적에서 진입·탈출하는 하전입자의 운동에너지 합이 같은 상태. 이 조건이 성립해야 Dose ≈ Kerma. 얇은 디텍터에서 깨짐 → PIT-001

KERMA

Kinetic Energy Released per unit Mass

비하전입자(감마, 중성자)가 물질과 상호작용하여 하전입자에 전달한 운동에너지의 질량당 합. 단위: Gy. Dose와 다름 → PIT-003

감쇠계수 (μ)

Linear Attenuation Coefficient

물질 단위 두께당 감마선 강도 감소율. I = I₀·e^(-μx). μ_en과 혼동 주의 → PIT-004

질량에너지흡수계수 (μ_en/ρ)

Mass Energy-Absorption Coefficient

감마선 에너지 중 실제 물질에 흡수되는 비율. KERMA→Dose 변환에 사용. 고Z 물질에서 에너지 의존성 큼 → PIT-019

빌드업 팩터

Buildup Factor

차폐체에서 산란선에 의한 선량 증가 비율. B = (직접선+산란선)/직접선. 에너지·물질·두께 의존 → PIT-027 LT-09

반가층 (HVL)

Half-Value Layer

방사선 강도를 절반으로 줄이는 차폐체 두께. n_HVL = ln(2)/μ. narrow-beam vs broad-beam 차이 → PIT-008

평균자유행로 (MFP)

Mean Free Path

방사선이 물질 내에서 상호작용 없이 평균적으로 진행하는 거리. λ = 1/μ. 차폐 두께 설계의 기본 단위.

역자승법칙

Inverse Square Law

점선원의 방사선 강도는 거리의 제곱에 반비례. I ∝ 1/r². 디텍터 크기가 거리 대비 클 때 보정 필요 → PIT-006

선량 측정 (6)

흡수선량

Absorbed Dose [Gy]

물질 단위 질량에 실제 흡수된 에너지. 1 Gy = 1 J/kg. MC 시뮬레이션의 기본 출력 물리량.

등가선량

Equivalent Dose [Sv]

흡수선량 × 방사선가중인자(Wr). 방사선 종류별 생물학적 효과 차이를 반영. H = D × Wr.

유효선량

Effective Dose [Sv]

등가선량 × 조직가중인자(Wt)의 합. 전신 피폭의 확률적 영향 평가. 법적 선량한도의 기준.

주변선량당량 H*(10)

Ambient Dose Equivalent

ICRU 구 30cm 깊이 10mm에서의 선량당량. 환경 모니터링의 실용량. ICRP-74 변환계수 사용 → ISS-020 LT-04

공기커마율

Air Kerma Rate [Gy/h]

공기 중 단위 시간당 KERMA. 감마선 선량 평가의 기본. 감마상수로부터 계산: K̇ = Γ·A/r².

방사선가중인자 (Wr)

Radiation Weighting Factor

방사선 종류별 생물학적 유효성 인자. 감마/베타=1, 알파=20, 중성자=에너지 의존(5~20). 단일 Wr의 한계 → PIT-024

핵물리 (5)

자발핵분열

Spontaneous Fission

외부 자극 없이 핵이 자발적으로 분열하는 현상. Cf-252의 주요 중성자 방출 메커니즘. 평균 3.77개 중성자/분열.

감마상수 (Γ)

Gamma Constant [R·m²/Ci·h]

단위 활동도 선원에서 1m 거리의 조사선량률. 핵종 고유값. K̇ = Γ·A/r² 관계. QSA 매뉴얼 참조 데이터.

붕괴당감마수

Gammas per Decay

핵붕괴 1회당 방출되는 감마선 수. GPS 스펙트럼 정의 시 intensity 비율 결정에 핵심 → PIT-005

다중도

Multiplicity

핵분열 1회당 방출되는 입자(중성자) 수의 분포. Cf-252: ν̄=3.77. GPS에서 반영 필요 → PIT-025

동위원소 농축

Isotope Enrichment

특정 동위원소의 비율을 자연 존재비 이상으로 높이는 처리. LT-21에서 농축 물질 지원 구현.

GEANT4 (6)

GPS

General Particle Source

GEANT4의 범용 입자 소스 모듈. 에너지 스펙트럼, 각도 분포, 공간 분포 정의. User Histogram 모드로 이산 스펙트럼 구현.

Physics List

물리 프로세스 목록

시뮬레이션에 포함할 물리 과정의 집합. QGSP_BERT_HP (중성자), em_standard_opt4 (전자기) 등. 선택이 결과에 큰 영향.

Production Cut

2차 입자 생성 문턱값

2차 입자 생성의 최소 에너지/거리 문턱값. 너무 높으면 저에너지 입자 누락 → PIT-010

Variance Reduction

분산 감소 기법

MC 시뮬레이션의 통계적 효율을 높이는 기법. Importance biasing, weight window 등. 부적절 사용 시 편향 → PIT-007

Importance Biasing

중요도 편향 기법

관심 영역의 입자 수를 통계적으로 증가시키는 VR 기법. 차폐 투과 문제에서 핵심. 중성자 적용 시 주의 → PIT-023

Sensitive Detector

감응 검출기

GEANT4에서 에너지 침적을 기록하는 논리 볼륨. Hit 생성 → 선량 계산. 크기·위치가 물리 결과에 직접 영향.

소프트웨어 (5)

CLI/GUI 하이브리드

Command-Line / GUI Hybrid

ShieldDose 아키텍처: Qt5 GUI가 GEANT4 CLI 바이너리를 spawn하여 결과 수집. GUI→매크로 생성→CLI 실행→결과 파싱.

N-layer 기하

N-Layer Geometry

다층 차폐 구조. 선원→[차폐1|차폐2|...|차폐N]→디텍터. 각 레이어별 물질·두께 독립 설정. 빌드업 독립곱 주의 → PIT-028

STL 임포트

STL Geometry Import

CAD 소프트웨어(OnShape 등)에서 내보낸 STL 메시를 GEANT4 기하로 변환. 메시 품질이 물리 정확도에 영향 → PIT-026

CMake 빌드

Cross-Platform Build System

GEANT4 및 Qt5 프로젝트의 빌드 시스템. find_package로 의존성 해결. 데이터 동기화 주의 → ISS-063

궤적 추적

Trajectory Tracking

입자의 이동 경로를 기록하여 시각화·분석. 산란선 기여 분석에 활용 → LT-05

표준/규제 (5)

ICRP-74

ICRP Publication 74 (1996)

외부 방사선에 대한 변환계수 제공. 플루언스→H*(10) 변환에 사용. ShieldDose에서 icrp74_photon.csv / icrp74_neutron.csv로 구현.

ICRP-116

ICRP Publication 116 (2010)

ICRP-74 업데이트. 확장된 에너지 범위, 추가 입자 타입. 향후 중성자 에너지별 변환에 활용 예정.

ANS 6.4.3

American Nuclear Society Standard

감마선 빌드업 팩터 데이터 표준. 에너지(0.015~15 MeV) × MFP(1~40) 격자. ShieldDose에서 CSV로 구현 → LT-09

NIST XCOM

NIST Photon Cross Section Database

광자 상호작용 단면적 데이터베이스. μ, μ_en 참조값 제공. 경험식 검증의 기준 → PIT-017

QSA Global

QSA Global (880 Series Manual)

방사선원 제조사. 880 시리즈 매뉴얼에서 감마상수, HVL 데이터 참조. Cs-137 Pb HVL 데이터 오류 발견 → PIT-015
